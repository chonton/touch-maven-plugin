touch:touch

Full name:

org.honton.chas:touch-maven-plugin:0.0.1:touch

Description:

Change modification time of specified files.

Attributes:

  • Requires a Maven project to be executed.

Required Parameters

Name Type Since Description
<files> FileSet - A fileSet selecting files and directories to touch.
<modificationTime> String - Modification timestamp to use. If numeric, the number of seconds in the Unix Epoch. If non-numeric, an ISO8601 timestamp.
User property is: touch.time.

Optional Parameters

Name Type Since Description
<skipTouch> boolean - Skip touching
Default value is: false.
User property is: touch.skip.

Parameter Details

<files>

A fileSet selecting files and directories to touch.
  • Type: org.apache.maven.shared.model.fileset.FileSet
  • Required: Yes

<modificationTime>

Modification timestamp to use. If numeric, the number of seconds in the Unix Epoch. If non-numeric, an ISO8601 timestamp.
  • Type: java.lang.String
  • Required: Yes
  • User Property: touch.time

<skipTouch>

Skip touching
  • Type: boolean
  • Required: No
  • User Property: touch.skip
  • Default: false
